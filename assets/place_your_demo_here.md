# Folder Assets Demo

Gunakan folder ini untuk meletakkan rekaman video demo localhost Anda.

### Cara Menggunakan:
1. Rekam video jalannya aplikasi Anda di localhost menggunakan tool perekam layar (seperti OBS Studio, Windows Game Bar `Win + Alt + R`, atau CapCut).
2. Simpan video tersebut di folder ini dengan nama **`demo.mp4`** (atau format `.gif`).
3. Setelah Anda melakukan push ke GitHub, video tersebut akan otomatis muncul dan bisa diputar langsung di halaman depan `README.md` repositori GitHub Anda!
